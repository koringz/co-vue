## 目录结构 

**covu组件结构说明**
```
  |--/build
  |--config
  |--/src/
  |	|--/assets/
  |	|--/component/
  |		|-- index.js
  |		|-- template.vue
  |		|-- /other component files
  |	  |-- example.vue
  |	   	|-- 注：使用co-component-name定义vue组件
  |	   	|-- 注：使用co-[component-name]定义组件class类名
  |	   	|-- 注：使用co-style-[other-name]定义其他class类名
  |--/pages/
  |	|-- /componentname
  |		|-- componentname.vue
  |--/router 路由组件
  |	|-- index.js # 路由入口
  |--/static
  |	|-- loading.js
  |--package.json
  |--index.html
  |--.gitignore
  |--.babelrc
  |--README.md
